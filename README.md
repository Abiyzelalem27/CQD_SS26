

# Computational Quantum Dynamics SS2026

This repository contains the codebase for the course **"Computational Quantum Dynamics" (Summer Semester 2026)**.

## Weekly Programming Exercises 

### Exercise 1: Single Particle in a 1D Potential
- Harmonic oscillator eigenfunctions (analytical solutions)
- Numerical solution using finite differences
- Matrix diagonalization using `numpy.linalg.eigh`
- Comparison between analytical and numerical eigenstates
- Convergence analysis with grid size

## Topics Covered
- Analytical vs numerical eigenfunctions  
- Grid discretization effects  
- Harmonic oscillator spectrum  
- Double-well potential and tunneling effects  
- Normalization of wavefunctions  
- Convergence of eigenvalues and eigenfunctions  

## Key Concepts
- Eigenvalue problems in quantum mechanics  
- Discretization of differential operators  
- Stability and convergence of numerical methods  
- Physical interpretation of quantum states
